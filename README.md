# Mi Portfolio

Portfolio personal como desarrollador de aplicaciones multiplataforma, construido con React y Vite.

🔗 **Web:** https://mi-portfolio.vercel.app

## Tecnologías

- React
- Vite
- Tailwind CSS
- lucide-react (iconos)

## Desarrollo local

```bash
npm install
npm run dev
```

El proyecto se abre en `https://mi-portfolio-pied.vercel.app/`.

## Despliegue

Desplegado en Vercel. Cada push a la rama `main` actualiza la web automáticamente.
